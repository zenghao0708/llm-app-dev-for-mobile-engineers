# 书名页

书名：AI Coding 编程专家
副标题：面向移动端工程师的 Agent 编程、工程上下文与自动化实践
系列定位：大模型应用开发三部曲第二册
建议分类：计算机 / 人工智能 / 软件工程 / 移动开发 / 开发者工具
目标读者：iOS 工程师、Android 工程师、Flutter/React Native 工程师、移动端架构师、测试开发工程师、研发效能工程师

## 内容简介

本书面向已经具备移动端工程经验的开发者，系统讲解 AI coding 从代码补全、聊天式编程到 agentic coding 的技术演进，并围绕 Prompt Engineering、Context Engineering、Harness Engineering 和 Loop Engineering 四类能力，建立一套可落地、可验证、可治理的 AI 编程方法。

全书不只介绍 Claude Code、Codex 等工具如何使用，更关注它们背后的工程模式：如何让 agent 理解仓库，如何提供合适上下文，如何设置权限和沙箱，如何让 agent 运行测试和检查，如何把多轮修改收敛到可审查的 Pull Request，如何在 iOS、Android、Flutter 和 React Native 项目中使用 AI coding 提升交付质量。

## 本书边界

本书不承诺某个具体工具永远领先，也不把 AI coding 简化成“提示词模板大全”。工具会快速变化，但仓库结构、上下文组织、测试闭环、安全边界和团队协作方法会长期存在。本书的目标是帮助读者掌握这些更稳定的工程能力。
