# 书名页

书名：大模型应用开发快速入门  
副标题：Prompt、RAG、Agent 与工程实践  
建议分类：计算机 / 人工智能 / 移动开发 / Python / App 工程实践  
目标读者：iOS 工程师、Android 工程师、Flutter/React Native 工程师、移动端架构师、移动测试工程师、AI 应用开发初学者  

## 内容简介

本书面向具有移动端开发基础的读者，系统介绍大模型应用开发的核心概念、常见架构和移动端工程实践。全书以 Python 服务端示例为主要代码主线，同时结合 iOS、Android、Flutter 和 React Native 的接入场景，从大模型调用、Prompt 设计、结构化输出、工具调用讲起，逐步进入 Embedding、向量检索、RAG、Agent、多模态输入、流式 UI、评测、安全、成本控制和上线运维等主题。

本书强调工程落地，而不是模型训练理论。读者不需要具备深度学习背景，只要理解 HTTP、JSON、数据库和基本编程方法，就可以跟随示例构建可运行的大模型应用。

## 封面建议

主视觉采用简洁白底技术教材风格，以 Mobile App -> API Gateway -> RAG -> Agent -> LLM 的架构图作为视觉中心。封面留白充足，突出书名和副标题，弱化装饰性元素。
