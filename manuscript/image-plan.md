# 全书高清配图规划

## 图片类型

1. 原理图：使用 SVG 矢量图，解释概念和架构。
2. 流程图：展示调用链路、RAG 流程、Agent 循环和评测闭环。
3. 截图：展示开发环境、运行结果、接口调试、移动端页面。
4. 代码图：用于展示关键代码和输出结果，不替代正文代码。
5. 项目图：展示综合项目的页面、目录结构和部署拓扑。
